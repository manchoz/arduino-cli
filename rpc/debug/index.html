    DebugReq¶

    The top-level message sent by the client for the Debug method. Multiple DebugReq messages can be sent but the first message must contain a DebugReq message to initialize the debug session. All subsequent messages must contain bytes to be sent to the debug session and must not contain a DebugReq message.

    Content must be either a debug session config or data to be sent.

    Field Type Label Description
    debugReq DebugConfigReq Provides information to the debug that specifies which is the target. The first StreamingOpenReq message must contain a DebugReq message.
    data bytes The data to be sent to the target being monitored.
    send_interrupt bool Set this to true to send and Interrupt signal to the debugger process
